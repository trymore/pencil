pencil

jQueryプラグインではありません。

jQueryをクラスとして継承することのできるViewクラス、それを継承したしたUIのクラス群、ユーティリティから成るライブラリです。

依存ライブラリ

  • jQuery v1.11.0〜 or v2.1.0〜
  • Lodash v2.4.0〜

フェッチ

bower install jquery lodash pencil

ビルド

browserifyでビルドすることを前提としています。
