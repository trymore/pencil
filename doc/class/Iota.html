Class: Iota

Defined in: models/iota.coffee

Overview

主にフラグを作る際に使用するユーティリティです。

Examples:

10進数のフラグを生成する 

  iota = require('pencil/models/iota')()
  a = iota() # 0
  b = iota() # 1
  c = iota() # 2
  d = iota() # 3
  e = iota() # 4

2進数のフラグを生成する 

  iota = require('pencil/models/iota')()
  a = 1 << iota() # 0 (0000)
  b = 1 << iota() # 1 (0001)
  c = 1 << iota() # 2 (0010)
  d = 1 << iota() # 4 (0100)
  e = 1 << iota() # 8 (1000)
