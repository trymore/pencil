Class: Sprite

Defined in: views/sprite.coffee
Inherits: View

Overview

スプライトアニメーションを管理するクラスです。

Examples:

同じViewに対して動き毎に別のSpriteを作ります。 

  run = new Sprite '.foo'
  run.setRange 0, 10
  walk = new Sprite '.foo'
  walk.setRange 11, 20
  run.play()
  run.on Sprite.EVENT_LAST_FRAME, ->
    walk.play()

Variables Summary

EVENT_LAST_FRAME = 
'sprite.lastFrame'

最終フレームの終了時に発火するイベントです。

EVENT_COMPLETE_REPEAT = 
'sprite.completeRepeat'

指定のリピート回数が完了した時に発火するイベントです。 永久にリピートする場合は発火しません。
